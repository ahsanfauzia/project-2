# Backend API Development

## Overview

This project is a simple REST API developed using Node.js and Express.js. It demonstrates backend development concepts including routing, request handling, validation, and JSON responses.

## Features

* REST API
* GET Endpoint
* POST Endpoint
* JSON Request Handling
* Basic Input Validation
* Express Server

## Technologies Used

* Node.js
* Express.js
* CORS

## API Endpoints

### Home

GET /

### Get Users

GET /users

### Create User

POST /users

## How to Run

Install dependencies

```
npm install
```

Run the server

```
node server.js
```

The server runs on:

```
http://localhost:5000
```
